# Grokking Embeddings

A small Python repository demonstrating embedding matrix initialization for text models.

## Files

- `Embeddings.py`: creates a random embedding matrix using NumPy.

## Usage

1. Install NumPy:

   ```bash
   pip install numpy
   ```

2. Run the script or import `EEmbeddings` from `Embeddings.py`.

## Notes

This repository is intentionally minimal and focused on the core embedding initialization utility.
